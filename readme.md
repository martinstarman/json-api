# stuckapi

Create read only API from your data with one command.

## Usage

- Add your data in json files to ```data``` dir.
- File name equals resource name, e.g. ```users.json``` equals ```/users```
- Run ```npm start```.
- Go to ```localhost:3000/<your-api-endpoint>```

### Run server

```npm start```

### Run tests
```npm test```
